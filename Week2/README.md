# Week 2 - Tesla ML Pipeline Project  
 
End-to-End Machine Learning Pipeline for Tesla EV Deliveries, Production Analytics, and Forecasting (2015-2025) 
 
## Overview 
This project implements a complete ML pipeline using Tesla EV sales and production data. 
 
## Features 
- Data Preprocessing 
- Exploratory Data Analysis 
- Feature Engineering 
- Linear Regression 
- Random Forest Regression 
- Hyperparameter Tuning 
- ARIMA Forecasting 
 
## Technologies Used 
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Statsmodels
